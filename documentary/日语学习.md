## 语法

### 第1课  わたしは　田中です

| 语法                     | 意思     | 解析                                 | 例句                             |
| ------------------------ | -------- | ------------------------------------ | -------------------------------- |
| 甲は　乙です             | 甲是乙   |                                      | わたしは　田中です               |
| 甲は　乙では　ありません | 甲不是乙 | “です”的否定形式是“では　ありません” | 田中さんは　学生では　ありません |
|                          |          |                                      |                                  |
